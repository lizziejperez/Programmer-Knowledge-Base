## Purpose

These notes are designed to:

- Reinforce core algorithmic thinking
- Compare performance tradeoffs across approaches
- Support interview preparation and problem solving
- Provide quick-reference material for projects and review

## Structure

Files in this folder generally include:

- Overview or requirements
- Time complexity
- Algorithm descriptions
- Pseudocode or step-by-step logic
- Tradeoffs and use cases
- Interview-relevant patterns when useful

Some files group multiple related algorithms together, such as sorting and searching.
