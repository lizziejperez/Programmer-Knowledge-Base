# Programming Languages

## Types of Programming Languages

### 1. Object-Oriented Programming (OOP) Language

Based on objects

Objects can contain data and code

- Data in the form of fields
- Code in the form of procedures

### 2. Functional Programming Language
Programs are constructed by applying and composing functions
Functions are variables

### 3. Procedural Programming Language
These languages execute code in a sequential manner, following a set of instructions step-by-step

### 4. Scripting Programming Language
These languages are often interpreted and used for automating tasks or creating interactive web applications

### 5. Logic Programming Language
These languages use logical rules and facts to solve problems

## Programming Language Typing
### Dynamic v.s Static
**Dynamic Typing**:
Type checking occurs at runtime

**Static Typing**:
Type checking occurs before any code is run

### Implicit v.s. Explicit
**Latent/Implicit Typing**:
Types are associated with the values (no need to explicitly give a type to a variable)

**Explicit Typing**:
Need to explicitly give a type to a variable
