# Formatting Guidelines

## Headers
- Use `#` for file title
- Use `##` for main sections
- Use `###` for subsections
- Use `####` for minor subsections (avoid going deeper)

## Code
### Blocks
Use triple backticks with language specified.
### In-line
```
The `code` is in-line.
```
The `code` is in-line.

## File Naming
- lowercase
- hyphen-separated
- no spaces

## Math

### In-line math:

Make sure you use `$ ... $` for inline math

Example:
```
$ \mu = \frac{1}{n} \sum_{i=1}^{n} x_i $
```
$ \mu = \frac{1}{n} \sum_{i=1}^{n} x_i $

### Block math:

Make sure you use `$$ ... $$` for block equations

Example:
```
$$
\bar{x} = \frac{1}{n} \sum_{i=1}^{n} x_i
$$
```
$$
\bar{x} = \frac{1}{n} \sum_{i=1}^{n} x_i
$$
