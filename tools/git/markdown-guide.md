## Headers
- Use `#` for file title
- Use `##` for main sections
- Use `###` for subsections
- Use `####` for minor subsections (avoid going deeper)

## Math

### Block math:

Use `$$ ... $$` for block equations

Example:
```
$$
\bar{x} = \frac{1}{n} \sum_{i=1}^{n} x_i
$$
```
$$
\bar{x} = \frac{1}{n} \sum_{i=1}^{n} x_i
$$
