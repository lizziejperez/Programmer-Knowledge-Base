# Data Analysis Workflow

A typical data analysis process involves collecting, cleaning, analyzing, and presenting data to support decision-making.

## Steps

### 1. Data Collection

Sources:

* Databases (SQL)
* CSV / Excel files
* APIs

### 2. Data Cleaning

Common tasks:

* Handling missing values
* Removing duplicates
* Fixing inconsistent formats
* Filtering invalid data

Example (Python):

```
df = df.dropna()
df = df.drop_duplicates()
```

### 3. Data Transformation

Prepare data for analysis:

* Grouping and aggregation
* Creating new columns
* Converting data types

### 4. Analysis

Explore patterns:

* Averages, counts, distributions
* Trends over time
* Relationships between variables

### 5. Visualization

Communicate insights using:

* Charts (bar, line, scatter)
* Dashboards (Power BI, Tableau)

### 6. Reporting

Present findings clearly:

* Summaries
* Dashboards
* Business insights

## Key Idea

The goal is not just analysis, but **supporting decision-making**.
